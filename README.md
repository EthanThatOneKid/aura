# aura-maxxing

[![skills.sh](https://skills.sh/b/EthanThatOneKid/aura-maxxing)](https://skills.sh/EthanThatOneKid/aura-maxxing)

A skill that keeps responses high-signal, concise, present, and honest.

## Usage

```bash
npx skills add https://github.com/ethanthatonekid/aura-maxxing --skill aura-maxxing
```

This adds the aura-maxxing skill to your project. Once installed, aura-maxxing will encourage concise, high-signal communication from your coding agent.

## What it does

- Encourages short, direct responses
- Reduces unnecessary verbosity and filler
- Keeps the agent present and grounded
- Promotes honesty and clarity over speculation
